Cài đặt kết nối

Anh Đậu cần access token của tài khoản YouPass để trả lời đúng theo bài học của em.

  1. Vào e-learning.youpass.vn, đăng nhập tài khoản của em.
  2. Mở DevTools (F12) → tab Network → gửi 1 câu hỏi cho chatbot trên web.
  3. Tìm request chat/stream → copy header x-youpass-token → dán vào ô dưới.

Token thường hết hạn sau ~3 ngày, khi bot báo lỗi thì lấy lại token mới nhé. Nếu admin đã cấu hình sẵn token trên server (biến môi trường), em có thể bỏ qua phần này.

2 kênh này thực chất gọi cùng 1 API nhưng giả origin khác nhau — nghi vấn backend phản hồi khác nhau tuỳ trang nhúng widget (vd. trang chủ có thể không tự bám lesson context). Đổi thử nếu kênh hiện tại cho kết quả không như ý.

Nếu thấy bot cứ bám theo một bài Writing/bài học cũ dù bạn hỏi chuyện khác: bấm "Cuộc trò chuyện mới" (nút ＋ trên header) để xoá lịch sử hội thoại, hoặc tự nhập nội dung vào đây để ép bot theo đúng ngữ cảnh bạn muốn.
